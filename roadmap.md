## **TypeScript Task Tracker Learning Roadmap**

### 1. **Define Task Structure**
- **Concept:** Types & Interfaces  
  - Learn how to define custom types and interfaces for objects.
  - **Task:** Create a `Task` interface/type with properties like `id`, `title`, `completed`.

### 2. **Create and Manage Tasks**
- **Concept:** Functions  
  - Learn to write functions with typed parameters and return values.
  - **Task:** Implement functions to add, edit, and delete tasks.

### 3. **Toggle Task Completion**
- **Concept:** Function Types & Type Narrowing  
  - Practice updating object properties and using function types.
  - **Task:** Write a function to mark a task as completed or not.

### 4. **Filter Tasks**
- **Concept:** Generics  
  - Learn to write generic functions for reusability.
  - **Task:** Create a generic filter function to show all, completed, or pending tasks.

### 5. **Organize with Classes**
- **Concept:** Classes  
  - Learn how to use classes to encapsulate logic and state.
  - **Task:** Create a `TaskManager` class to manage the task list and related operations.

### 6. **TypeScript in Practice**
- **Concept:** Putting It All Together  
  - Combine interfaces, functions, generics, and classes in a small app.
  - **Task:** Build a simple CLI or web-based task tracker using the above concepts.

---

**Tip:**  
Work through each step in order, practicing the TypeScript concept as you implement the feature!
